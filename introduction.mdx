### Overview

Murf AI is a [text-to-speech](https://murf.ai/text-to-speech) platform that lets you generate lifelike AI voiceovers from text in minutes. With the Murf API, you can integrate versatile voice generation capabilities into your products, applications, and workflows to enhance user experiences and enable innovative features.

***

### Getting Started

<Accordion title="Step 1: Authenticate with the API">
To start using the Murf API, you’ll need an API key to authenticate your requests. Follow these steps:
Create an Account: If you don’t have a Murf account yet, sign up to access the API dashboard.
Log In: Once you’ve created your account, log in to your dashboard.
Generate API Key: Navigate to the API documentation page, and on the right side, click the Create API Key button (as shown in the image below) to generate your key.
All requests to the Murf API must include an Authorization header with your API key. Be sure to include this header manually in each request for authentication. You can manage, restrict, or delete API keys as needed for enhanced security.
. For more, check out [authentication docs](https://murf.ai/api/reference#authentication).
</Accordion>

<Accordion title="Step 2: Synthesize Your First Audio">
  Send a POST request to `/v1/speech/generate` with your text and desired voice parameters. For detailed examples, see the [API Reference](https://murf.ai/api/reference#synthesize-audio).
</Accordion>

### Key Features

| Feature            | Description                                                                                                          |
| ------------------ | -------------------------------------------------------------------------------------------------------------------- |
| **Audio Quality**  | Sampling rates include 8Khz, 24Khz, 44.1Khz, and 48Khz.                                                              |
| **Voice Options**  | Offers 120+ realistic voices in 8 languages and 20+ styles, with flawless native accents.                            |
| **Voice Styles**   | Includes storytelling, meditative, newscast formal, promo, casual, narration, and more.                              |
| **Languages**      | Supports English (US/Canada, UK, Australia, India), French, German, Spanish, Italian, Hindi, Portuguese, and Polish. |
| **Speed & Pitch**  | Control speed and pitch for tailored outputs.                                                                        |
| **Output Formats** | Supports MP3, WAV, FLAC, ALAW, ULAW, and Base64 encoding.                                                            |

### Troubleshooting

#### Token Expiration

Auth tokens generated by the [Create Auth Token](https://murf.ai/api/reference#create-auth-token) endpoint expire 30 minutes after creation. Ensure token regeneration to prevent disruptions.
