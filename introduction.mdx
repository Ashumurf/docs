### Overview

Murf AI is a [text-to-speech](https://murf.ai/text-to-speech) platform that lets you generate lifelike AI voiceovers from text in minutes. With the Murf API, you can integrate versatile voice generation capabilities into your products, applications, and workflows to enhance user experiences and enable innovative features.

### Key Features

| Feature            | Description                                                                                                          |
| ------------------ | -------------------------------------------------------------------------------------------------------------------- |
| **Audio Quality**  | Sampling rates include 8Khz, 24Khz, 44.1Khz, and 48Khz.                                                              |
| **Voice Options**  | Offers 120+ realistic voices in 8 languages and 20+ styles, with flawless native accents.                            |
| **Voice Styles**   | Includes storytelling, meditative, newscast formal, promo, casual, narration, and more.                              |
| **Languages**      | Supports English (US/Canada, UK, Australia, India), French, German, Spanish, Italian, Hindi, Portuguese, and Polish. |
| **Speed & Pitch**  | Control speed and pitch for tailored outputs.                                                                        |

### Troubleshooting

#### Token Expiration

Auth tokens generated by the [Create Auth Token](https://murf.ai/api/reference#create-auth-token) endpoint expire 30 minutes after creation. Ensure token regeneration to prevent disruptions.
